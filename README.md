# 📊 Dashboard Analítico de Vendas Globais

## 📌 Sobre o Projeto
Este projeto consiste na criação de um dashboard interativo no Power BI para análise de vendas globais.  
A base de dados utilizada já estava previamente tratada (ETL), com foco na exploração, visualização e geração de insights.

---

## 🎯 Objetivo
Analisar o desempenho de vendas considerando diferentes dimensões, como:
- Categoria de produtos
- País
- Segmento de clientes
- Descontos aplicados

O objetivo é transformar dados em informações úteis para apoiar a tomada de decisão.

## 🔍 Análises Realizadas

- Distribuição das vendas por categoria  
- Volume de pedidos por país e prioridade  
- Média de desconto por subcategoria  
- Análise geográfica das vendas  
- Filtros interativos por ano, segmento e país  

---

## 💡 Principais Insights

- A categoria **Tecnologia** representa a maior participação nas vendas totais  
- Os **Estados Unidos** concentram o maior volume de pedidos  
- Algumas subcategorias apresentam **altas médias de desconto**, o que pode impactar a lucratividade  
- Há variação no desempenho de vendas entre regiões e segmentos
